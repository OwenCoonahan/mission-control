## API ROUTES TO BUILD
- GET/POST /api/tasks - list and create tasks
- PUT/DELETE /api/tasks/[id] - update and delete tasks
- GET/PUT /api/goals - list and update goals
- GET /api/activity - fetch recent activity from session logs
- GET /api/cron - fetch cron job schedule

## TASK SCHEMA
```json

  "id": "string",
```
