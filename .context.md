# Mission Control - Project Context

## What This Is
A personal command center dashboard for Owen, with Sir Henry (AI assistant) as the engine.
NOT a mockup. A real, usable tool.

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS
- shadcn/ui components
- File-based JSON storage (data/ folder)
- TypeScript

## Design Principles
1. **Dark theme** - zinc-950 background, zinc-900 cards
2. **Functional first** - every button does something
3. **Real data** - no mocks, everything persists
4. **Clean & minimal** - information dense but not cluttered
5. **Test before commit** - verify UI renders correctly

## Color System
- Background: `bg-zinc-950` (#09090b)
- Cards: `bg-zinc-900` (#18181b)
- Borders: `border-zinc-800` (#27272a)
- Hover: `hover:border-zinc-700`
- Text primary: `text-zinc-100`
- Text secondary: `text-zinc-400`
- Accent (violet): `text-violet-400`, `bg-violet-500`
- Success (green): `text-emerald-400`
- Warning (amber): `text-amber-400`
- Danger (red): `text-red-400`

## Layout Structure
```
┌─────────────────────────────────────────────────┐
│ Sidebar (w-64)  │  Main Content (flex-1)        │
│                 │                               │
│ - Logo          │  Header (sticky)              │
│ - Nav Items     │  ├─ Page Title                │
│ - User Avatar   │  └─ Actions                   │
│                 │                               │
│                 │  Content Area (scrollable)    │
│                 │                               │
└─────────────────────────────────────────────────┘
```

## Pages
1. **Dashboard** (`/`) - Today's focus, goals, stats
2. **Tasks** (`/tasks`) - Kanban board
3. **Goals** (`/goals`) - Progress tracking
4. **Activity** (`/activity`) - Agent logs + cron jobs
5. **Memory** (`/memory`) - Browse workspace files

## Data Files
- `data/tasks.json` - All tasks
- `data/goals.json` - Goal progress
- Cron jobs: read from `~/.openclaw/cron/jobs.json`

## API Routes
- `GET/POST /api/tasks` - List/create tasks
- `PUT/DELETE /api/tasks/[id]` - Update/delete task
- `GET/PUT /api/goals` - Get/update goals
